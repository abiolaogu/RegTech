# RegTech Horizon: User Manual

## 1. Web Dashboard Guide
### 1.1 Logging In
1.  Navigate to `https://platform.regtech-horizon.com`.
2.  Enter your corporate email and password.
3.  **MFA**: Enter the 6-digit code sent to your Authenticator App.

### 1.2 Generating a Compliance Report
1.  Go to **"Reports"** in the left sidebar.
2.  Select **"New Report"**.
3.  Choose Jurisdiction (e.g., **NCC QoS**).
4.  Upload required data files (CSV/Excel).
5.  Click **"Generate"**.
6.  *Wait 2-5 minutes.* You will receive a notification when the PDF is ready.

### 1.3 Interpreting the "Three-Cs" Dashboard
- **Completeness**: % of required data fields present. *Target: 100%*.
- **Correctness**: % of data passing validation rules. *Target: >99%*.
- **Consistency**: Score based on cross-report variance. *Target: >95%*.

## 2. Mobile App Guide
### 2.1 Installation
- **iOS**: Download from TestFlight (Link provided by IT).
- **Android**: Download APK from SharePoint.

### 2.2 Biometric Login
- On first launch, enter username/password.
- Tap **"Enable FaceID"** when prompted.
- Subsequent logins will use biometrics.

### 2.3 Responding to Alerts
1.  Push Notification: "Critical: NCC Levy payment overdue."
2.  Tap notification to open **Alert Details**.
3.  Review the AI-generated explanation.
4.  Tap **"Acknowledge"** to mark as seen, or **"Escalate"** to notify a manager.
